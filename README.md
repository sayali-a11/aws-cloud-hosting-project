# ☁️ AWS Cloud Hosting Project

## 🚀 Live Cloud Hosting using AWS EC2, VPC & Apache Server

A professional cloud deployment project demonstrating secure and scalable website hosting using Amazon Web Services (AWS). This project showcases practical implementation of EC2 instances, Virtual Private Cloud (VPC) networking, Apache Web Server configuration, and remote server management using MobaXterm SSH client.

# 📌 Project Overview

This project demonstrates how to deploy a modern responsive website on a cloud infrastructure using AWS services. The website is securely hosted on an Ubuntu EC2 instance configured inside a Virtual Private Cloud (VPC) with Apache2 web server support.

The project covers:

* Cloud Infrastructure Setup
* VPC Networking
* EC2 Server Deployment
* Apache Web Server Configuration
* Secure SSH Remote Access
* Live Website Deployment

# 💻 Deployment Process

## Step 1 — AWS Account Setup

Created AWS Free Tier account.

## Step 2 — VPC Configuration

Configured custom VPC with public subnet and internet gateway.

## Step 3 — EC2 Launch

Launched Ubuntu EC2 instance with public IP.

## Step 4 — Security Group Configuration

Enabled:

* SSH access
* HTTP access

## Step 5 — Apache Installation

sudo apt update
sudo apt install apache2 -y

## Step 6 — Website Deployment

cd /var/www/html
sudo nano index.html

## Step 7 — Live Hosting

Website successfully deployed using EC2 public IP.
